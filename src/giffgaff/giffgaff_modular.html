⚠️ Important: OAuth Login Prerequisites

在点击下方"开始OAuth登录"按钮之前，您必须先完成以下步骤：

  1. 访问 giffgaff.com 并登录您的账户
  2. 确保登录状态有效（可以看到您的账户信息）
  3. 保持该标签页打开，然后返回本页面

为什么需要这样做？ Giffgaff 现在要求用户必须先在官网登录，才能通过 OAuth 授权。如果未登录就点击 OAuth 登录，将无法获取回调 URL。

Before clicking "Start OAuth Login" below, you MUST complete these steps first:

  1. Visit giffgaff.com and log in to your account
  2. Ensure your login session is active (you can see your account info)
  3. Keep that tab open, then return to this page

Why is this required? Giffgaff now requires users to be logged in on their website before OAuth authorization. Without prior login, you won't be able to capture the callback URL.

Cookie 登录
如何获取Cookie：
  1. 访问 giffgaff.com 并登录您的账户
  2. 打开开发者工具（Chrome：F12 或 右键→检查）
  3. 切换到 Application（应用）/ Storage（存储）标签页
  4. 在左侧选择 Cookies → https://www.giffgaff.com
  5. 复制右侧所有条目，拼成 name=value; name2=value2; ... 格式
或 在已登录的 giffgaff.com 页签，打开控制台(Console)粘贴以下代码并回车： 
;(() => {
  try {
    const out = document.cookie.split('; ').filter(Boolean).join('; ');
    console.log(out);
    if (navigator.clipboard && window.isSecureContext) {
      navigator.clipboard.writeText(out)
        .then(() => console.log('Cookie 已复制到剪贴板'))
        .catch(() => {});
    }
  } catch (e) {
    console.log('读取 Cookie 失败:', e);
  }
})();
安全提示：仅在可信设备上粘贴 Cookie。本工具不会存储您的 Cookie。
